# SentinelCore

SentinelCore is a next-generation enterprise security platform featuring a high-performance interface and 3D interactive elements. Built with [Next.js 16](https://nextjs.org), React 19, and Tailwind CSS v4.

## Features

-   **Modern Tech Stack**: Powered by Next.js 16 (App Router) and React 19.
-   **3D Visuals**: Integrated Spline 3D assets for immersive user experience.
-   **Responsive Design**: Fully responsive UI built with Tailwind CSS.
-   **Enterprise Grade**: Designed for scalability and performance.

## Getting Started

### Prerequisites

-   Node.js 18+ 
-   npm, yarn, pnpm, or bun

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/sentinel-core.git
    cd sentinel-core
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

-   `dev`: Runs the development server.
-   `build`: Builds the application for production.
-   `start`: Starts the production server.
-   `lint`: Runs ESLint to check for code quality issues.

## Docker

To build and run the application using Docker:

1.  Build the image:
    ```bash
    docker build -t sentinel-core .
    ```

2.  Run the container:
    ```bash
    docker run -p 3000:3000 sentinel-core
    ```
